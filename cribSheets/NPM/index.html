Getting started

  1. Setting up NPM
    • download & install NODEJS which comes with npm
      WTF is nodejs? Nodejs allows running JS outside of the browser
    • check NPM installation by running npm -v in the terminal
    • check NODE installation by running node -v in the terminal
    • update NODE by downloading the latest version from nodejs.org (will also update npm)
    • update only NPM by running in terminal npm install npm@latest -g
    • if running into permission errors while installing from terminal, use sudo in front of each command

  2. Start using NPM
    • when starting a project run npm init which creates a package.json file in the project directory.
      This file allows grabbing any package from npm

  3. Install packages
    • packages can be installed globally to be available anywhere on the computer(those with -g )
    • ex: to install Live Server globally run npm install -g live-server in the terminal. Then run live-server
    • packages can also be installed locally, to be available for a particular project
    • ex: to install lodash locally run npm install lodash in the the terminal, in the project's folder
