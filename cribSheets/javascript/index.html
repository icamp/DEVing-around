Destructuring

The idea of destructuring is to take an object or an array and and break it down into smaller objects, elements or smaller variables.

const alphabet = [ 'A', 'B', 'C', 'D', 'E', 'F' ]

const numbers = [ '1', '2', '3', '4', '5', '6' ]

Ex: get the first 2 elements in the 'alphabet' array:
const [first, second] = alphabet;
console.log(first) will return A which is the first item in the array

Skipping an element in the array is done using a comma (,)
Ex: const [one,, two] = alphabet;
console.log(two) returns 'C' since 'B' is skipped by the double ,,

To get the rest of the elements in the array, the spreak operator ( ... ) can be used:
const [one,, two, ...rest] = alphabet;
console.log(rest) returns ["D", "E", "F"]

With destructuring and spread it is possible to combine 2 arrays together:
const combinedArray = [...alphabet, ...numbers]
